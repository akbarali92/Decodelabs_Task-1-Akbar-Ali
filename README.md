# Project 1 – Data Cleaning & Preparation

## Overview
This project focuses on cleaning and preparing a raw e-commerce dataset for further analysis. The main objective was to identify and correct data quality issues to ensure the dataset becomes accurate, consistent, and analysis-ready.

This project was completed as part of a Data Analytics Internship Program.

---

# Objectives
- Handle missing values
- Remove duplicate records
- Standardize text formatting
- Verify numeric data consistency
- Correct date formatting
- Prepare clean dataset for analysis

---

# Tools Used
- Google Sheets
- Microsoft Excel

---

# Dataset Information
The dataset contains e-commerce order data including:
- Order ID
- Customer ID
- Product details
- Quantity
- Unit Price
- Total Price
- Payment Method
- Order Status
- Referral Source
- Coupon Codes

---

# Data Cleaning Tasks Performed

## 1. Handling Missing Values
- Identified blank cells in the `CouponCode` column
- Replaced missing values with `"No Coupon"`

## 2. Duplicate Verification
- Checked `OrderID` column for duplicate records
- Verified dataset consistency

## 3. Text Standardization
- Removed extra spaces
- Standardized text formatting using proper capitalization

## 4. Numeric Data Validation
Verified numeric columns including:
- Quantity
- UnitPrice
- TotalPrice
- ItemsInCart

Applied proper number formatting and decimal consistency.

## 5. Date Formatting
- Standardized date formats across the dataset

---

# Final Result
The dataset was successfully cleaned and prepared for analysis. All missing values were handled, duplicate records were verified, and formatting inconsistencies were corrected.

The final dataset is now:
- Accurate
- Consistent
- Organized
- Ready for analysis and visualization
